## Installation
- `git clone https://github.com/romanV7/rest.git`
- `cd rest && npm i`
- `bash firstrun.sh`
- `mysql -h 127.0.0.1 -P 3306 -u root -p test_db < src/database.sql`
- `npm run start`

## Notice
password for database is `root` 
before uploading files make sure that directory `uploads` is created
